# Query Set Go — SQL Developer Handbook

Welcome to **Query Set Go**, a comprehensive SQL knowledge base and developer handbook created by **A-Stack**. It is designed to take developers from absolute beginners to SQL masters and interview-ready database experts.

This handbook is built as a single-page, responsive, offline-first web application. It features comprehensive guides, visual interactive diagrams, curated checklists, and a built-in SQL playground.

---

## 🚀 Key Features

*   💻 **Interactive SQL Playground & Simulator Sandbox**
    *   Test queries instantly in the browser without installing database engines.
    *   Preloaded with mock database tables (`users` and `posts`) to practice queries on the spot.
    *   Toggle it anytime using the shortcut `Ctrl + \`` (backtick).
*   📖 **27 Structured Learning Modules**
    *   Covers SQL Basics, Filtering, Grouping, Advanced Joins, Window Functions, CTEs, Transactions, Normalization, and Query Optimization.
*   🔍 **Fuzzy Search Command Palette**
    *   Instantly query chapters and topics by pressing `Ctrl + K`.
*   📊 **Visual Interactive SVG Diagrams**
    *   Understand complex database relationships and SQL joins visually.
*   🌙 **Modern Responsive Theme Engine**
    *   Tailored with vibrant dark and light modes.
*   📑 **Local Bookmarks & Progress Tracking**
    *   Save pages for reference later with localStorage persistence.
    *   Features an active reading progress bar indicator.
*   🎓 **Interview Preparation & Cheatsheets**
    *   Includes curated interview questions with accordion answers, cheat sheets, curated resources, and project ideas.

## 🛠️ How to Run & Use Locally

Since the handbook is designed to be **offline-first** and fully client-side, running it is simple:

1.  Clone this repository or download the folder.
2.  Double-click `index.html` to open it directly in any modern web browser (Chrome, Edge, Firefox, Safari).
3.  Alternatively, serve it using any simple static file server:
    *   **Python:** `python -m http.server 8000`
    *   **Node.js:** `npx serve .`

## 🧭 Handbook Modules Overview

The handbook is divided into five core sections:

1.  **SQL Handbook Foundations:** Welcome, Introduction, Getting Started.
2.  **Core SQL Guide:** SQL Basics, Databases, Tables, Queries, Filtering, Sorting, Functions, Grouping.
3.  **Relational Analytics:** Joins, Subqueries.
4.  **Advanced Database Concepts:** Views, Indexes, CTEs, Window Functions, Transactions, Constraints, Normalization, Optimization, Advanced SQL.
5.  **Preparation & Practice:** Interview Prep, Cheat Sheets, Practice Sites, Curated Resources, Project Ideas, SQL Roadmap.
